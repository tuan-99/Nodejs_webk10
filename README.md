## Clone dữ liệu
```
git clone https://github.com/tuan-99/Nodejs_webk10.git
```

### Chuyển nhánh đến nhóm của mình
```
git checkout -b [Tên Nhóm]
```

### add dữ liệu 
```
git add .
```

### commit dữ liệu
```
git commit -m "Nội dung commit"
```

### push dự liệu lên 
```
git push -u origin [Tên Nhóm]
```

### cập nhật dữ liệu 
git pull origin [Tên Nhóm]
